Define these models:

```python
class PromptConfig(BaseModel):
    prompt_mode: str
    prompts_file: str | None = None
    prompts_file_sha256: str | None = None
    few_shot_config: str | None = None
    few_shot_config_sha256: str | None = None
    prompt_prefix_sha256: str | None = None
    prompt_suffix_sha256: str | None = None

class RunManifest(BaseModel):
    schema_version: str = RUN_MANIFEST_SCHEMA_VERSION
    run_id: str
    created_at: datetime
    code_revision: dict[str, Any]
    python_version: str
    dependency_versions: dict[str, str]
    dataset_summary: dict[str, Any]
    prompt_config: PromptConfig
    provider: str
    model: str
    seed: int | None = None
    retry_policy: dict[str, Any] = Field(default_factory=dict)
    cost_control: dict[str, Any] = Field(default_factory=dict)
    output_paths: dict[str, str] = Field(default_factory=dict)
    ethics_scope: str = "offline authorized datasets only; no live service automation"

class AttemptRecord(BaseModel):
    schema_version: str = ATTEMPT_RECORD_SCHEMA_VERSION
    run_id: str
    attempt_id: str
    task_type: str
    puzzle_id: str
    attempt_index: int
    prompt_mode: str
    provider: str
    model: str
    parsed_answer: Any = None
    correct: bool
    error_category: str | None = None
    latency_ms: float | None = None
    tokens_in: int | None = None
    tokens_out: int | None = None
    cost_usd: float | None = None
    timestamp: datetime

class SummaryRow(BaseModel):
    schema_version: str = SUMMARY_ROW_SCHEMA_VERSION
    run_id: str
    provider: str
    model: str
    task_type: str
    n_attempts: int
    n_success: int
    pass_rate: float
```

Implement helper functions:
- `utc_now() -> datetime`
- `collect_code_revision() -> dict[str, Any]` using `git rev-parse HEAD` and a path-scoped dirty check that does not read `secrets.yaml`.
- `collect_dependency_versions(names: Iterable[str]) -> dict[str, str]` using `importlib.metadata.version()`.
- `sha256_file(path: str | Path | None) -> str | None` returning `None` for absent paths and the SHA-256 digest of the file bytes otherwise.
- `sha256_text(value: str | None) -> str | None` returning `None` for absent strings and the SHA-256 digest of the UTF-8 text otherwise.
- `revision_run_dir(output_root: str | Path, run_id: str) -> Path`.

Implement `RevisionArtifactWriter` with:
- `__init__(self, output_root: str | Path, run_id: str, *, overwrite: bool = False, resume: bool = False)`.
- Properties `run_dir`, `manifest_path`, `attempts_path`, `summary_csv_path`, `summary_json_path`.
- `write_manifest(self, manifest: RunManifest) -> Path`.
- `append_attempt(self, attempt: AttemptRecord) -> Path`.
- `iter_attempts(self) -> Iterator[AttemptRecord]`.
- `write_summaries_from_attempts(self) -> tuple[Path, Path]`.

`write_summaries_from_attempts()` must group attempts by `(run_id, provider, model, task_type)`, compute `n_attempts`, `n_success`, and `pass_rate = n_success / n_attempts`, and write both `summary.csv` and `summary.json`. Do not include raw secret config fields in any model.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| In-memory run config -> artifact files | Manifests and attempts become durable evidence consumed by later phases. |
| Attempt stream -> aggregate summary | Summary rows must derive from JSONL attempts instead of mutable aggregate state. |
| Existing output directory -> new run | Existing results must not be overwritten without explicit intent. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-03-01 | Information Disclosure | `RunManifest` | mitigate | Schema contains provider/model labels and metadata only, with no credential fields. |
| T-01-03-02 | Repudiation | `run_manifest.json` | mitigate | Include schema version, code revision, Python version, dependency versions, dataset summary, prompt config, seed, retry policy, and cost controls. |
| T-01-03-03 | Tampering | `attempts.jsonl` | mitigate | Append one validated `AttemptRecord` per line before summary derivation. |
| T-01-03-04 | Tampering | Existing run directory | mitigate | `RevisionArtifactWriter` refuses existing directories unless `overwrite=True` or `resume=True`. |
| T-01-03-05 | Repudiation | `summary.csv` and `summary.json` | mitigate | Derive summaries from `attempts.jsonl` and test row counts. |
</threat_model>
